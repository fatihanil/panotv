TVPano web uygulaması Denizli Mustafa Kaynak Anadolu Lisesi için Fatih Anıl tarafından tasarlanmış ve üretilmiş bir yazılımdır.
Okul ortak alanında yerleştirilmiş bir internet bağlantısı kurabilen bir smart TV ya da stick computer bağlanmış bir standart TV üzerinde koşmak üzere programlanmıştır.
Arka uçta MySQL veri tabanı ve SQL betikleri kullanılmıştır. Ön uç ise Bootstrap ve HTML ile yapılandırılmıştır.
Web uygulaması bir IIS veya Apache sunucu üzerinde koşacak mimaride tasarlanmıştır.
Lisanslı bir uygulama olup kullanılmak istenirse Fatih Anıl ile iletişime geçilebilir.
